# SmartSafe Backend

This repository contains the **backend** of the **SmartSafe project**, a software system designed to prevent fire hazards. The backend connects users to their devices through the mobile application and manages all related data and operations.

## Project Overview

**SmartSafe** is a mobile and embedded systems project aimed at:

- Monitoring devices to prevent fire hazards.
- Connecting users to their devices via a mobile application.
- Managing real-time notifications and alerts.

This repository is specifically for the **backend services** of the mobile app.

---

## Features

- User authentication and authorization
- Device management
- Real-time notifications
- Integration with mobile frontend
- Database management using Prisma and PostgreSQL

---

## Technologies Used

- **Node.js** & **NestJS** – backend framework
- **Prisma** – ORM for database interactions
- **PostgreSQL** – relational database
- **JWT** – authentication
- **TypeScript** – language for backend code
